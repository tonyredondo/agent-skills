# agent-skills

Repositorio para centralizar skills de agentes.

## Estructura

- `podcast-maker/`: skill para generacion de guion y audio de podcast.

## Convenciones

- Cada skill vive en su propia carpeta de primer nivel.
- Cada skill debe incluir su propia documentacion (`README.md` y/o `SKILL.md`).
- No subir archivos temporales, caches, checkpoints ni artefactos locales.

## Estado actual

Actualmente el repositorio incluye un skill: `podcast-maker`.
